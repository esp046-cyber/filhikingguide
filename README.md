# filhikingguide
Key Features:
4 Main Tabs: Mountains, Gear, Tips, and Emergency
Small font titles with 1% margins as requested
Filipino hiking theme with green/mountain colors
Mobile-optimized layout that works perfectly on phones
No 3D elements or search bar as specified

Popular Philippine mountains (Pulag, Apo, Pinatubo, Ugo) with elevation, difficulty, and best hiking seasons
Interactive gear checklist you can check off
Essential hiking tips specific to Philippine conditions
Emergency contact numbers (911, NDRRMC, Red Cross, etc.)
